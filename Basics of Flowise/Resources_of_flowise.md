## üîó Related Technology Documentation

### Core Technologies

#### LangChain
- **Official Docs:** https://python.langchain.com
- **Key Concepts:**
  - Chains: Sequential operations
  - Agents: Reasoning engines
  - Retrievers: Document search
  - Tools: External integrations
  - Memory: Conversation history

#### LangGraph
- **Documentation:** https://langchain-ai.github.io/langgraph/
- **Purpose:** Orchestration of complex agent workflows
- **Used in:** Sequential Agents architecture, multi-turn conversations

#### LLM Providers

| Provider | Pricing | URL |
|----------|---------|-----|
| **OpenAI** | $0.15/1K tokens (GPT-3.5) | https://openai.com |
| **Anthropic** | $0.80-24/1M tokens (Claude) | https://anthropic.com |
| **Google** | $0.075-0.30/1M tokens | https://cloud.google.com/generative-ai |
| **Together AI** | Open-source models | https://together.ai |
| **Ollama** | Free (local models) | https://ollama.ai |

### Vector Databases

#### Pinecone
- **URL:** https://pinecone.io
- **Free Tier:** 1 project, up to 100K vectors
- **Use Case:** Cloud-hosted embeddings, semantic search
- **Flowise Integration:** Built-in vector store node

#### Weaviate
- **URL:** https://weaviate.io
- **Free Tier:** Open-source, self-hosted
- **Use Case:** Hybrid search (keyword + semantic)
- **Flowise Integration:** Supported

#### Milvus
- **URL:** https://milvus.io
- **License:** Open-source
- **Use Case:** Scalable vector search
- **Flowise Integration:** Available

#### Postgres pgvector
- **URL:** https://github.com/pgvector/pgvector
- **Advantage:** Built on PostgreSQL, no separate database
- **Use Case:** Cost-effective for small-medium scale
- **Flowise Integration:** Supported
